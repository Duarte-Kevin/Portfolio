MY PROJECTS

PAPPR

PAPR is an educational web trading application designed for newbies to trade on the financial markets. The application is geared towards teachers and students, where teachers can create a room with a particular set of companies for students to trade virtual money. Technology used: [React, Redux, Socket.IO, IEX API, React-Vis, Material-UI, Sentiment, NodeJS, ExpressJS, Sequelize, FireBase, PostgresQL]
Github Website

FaceLogin

Application which allows users to register their personal information along with photos of their face. Users can then login by using facial recognition. Technology used: [Kairos API, JQuery, Sequelize, NodeJS, ExpressJS, PostgresQL]
Github Presentation

Ride Share Compare

I'm current working on an application which aggregates pricing estimates from various ride sharing applications ( Uber, Lyft ..etc). The objective of the application is to have an easy to view breakdown of price comparisons between various ride sharing services
